#ПРОФЕССИОНАЛЬНЫЕ ДОСТИЖЕНИЯ

Здравствуйте! Меня зовут Орлова Анастасия.

Это моё портфолио, созданное для поступления в магистратуру ИТМО на программу «Искусственный интеллект» (направление «Информатика и вычислительная техника»). 
Здесь собраны мои учебные и практические работы, которые демонстрируют навыки в тестировании ПО, работе с базами данных, языке SQL и программировании на Python.
Все представленные материалы — результат самостоятельного обучения и прохождения курсов повышения квалификации, а также курсов на онлайн-платформах.

---

#Навыки и инструменты

|Тестирование ПО| Составление тест-кейсов, чек-листов, баг-репортов, работа с Jira 
|API тестирование| Postman, тестирование REST API 
|Базы данных| SQL (продвинутый уровень), работа с СУБД, написание сложных запросов 
|Программирование| Python (основы, учебные скрипты) 
|Методологии| SDLC, STLC, написание отчётности о тестировании 

---

#Структура репозитория

| [`website-analysis/`] | Анализ сайта Яндекс.Музыка, определение его основных функиональных элементов  
| [`test-cases/`] | Тест-кейсы (6 позитивных, 4 негативных) для функционального и регрессионного тестирования 
| [`bug-reports/`] | Баг-репорты с описанием шагов, важности, серьёзности, приоритета, окружение 
| [`checklists/`] | Чек-листы для быстрой проверки ключевого функционала 
| [`testing-reports/`] | Отчет о тестировании по чек-листу
| [`data base/`] | Артефакты по созданию таблицы в БД MySQL и запросы к ней путем языка SQL
| [`sql-queries/`] | SQL-запросы разного уровня сложности 
| [`postman-collections/`] | Создание environment, создание 10 запросов разного вида, использование Collection Runner
| [`final-project/`] | Презентация итоговой аттестационной работы по курсу «Тестирование ПО»
| [`teacher-feedback/`] | Обратная связь преподавателя по курсу на все самостоятельные работы 
| [`python/`] | Учебные скрипты на Python
